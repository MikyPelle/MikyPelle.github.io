Purpose of the project

The project was developed for recreational purposes with the aim of following some
paths through a machine created both physically and virtually. Everything is based
on the use of the muse2 sensor which is used to detect brain waves and head
movement. As regards the physical part, the alphabot is piloted through the data
detected by muse2 and at the same time it is possible to view it on the screen
of the computer the path to take with the machine implemented using
Unreal Engine and Python.

Development timeline

1. Program development for receiving data from the Muse2 sensor

2. Installation and configuration of the Raspberry Pi operating system
on the Alphabot robot

3. Implementation of Client-Server communication between the computer
which runs the program and the aplhabot.

4. Graphic creation using Unreal-Engime 5 of the route on PC

5. Creation and implementation of the Python code to allow
the realization of the logic of the virtual game.
